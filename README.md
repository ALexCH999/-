# 🧮 Калькулятор логических функций на Python

## 📋 Описание

Графическое приложение на **Python + Tkinter** для построения **таблиц истинности** логических выражений.  
Позволяет вводить формулы с булевыми операторами и автоматически отображает все подвыражения и финальный результат.

---

## ⚙️ Поддерживаемые операторы

| Оператор | Описание           |
|----------|--------------------|
| `~A`     | НЕ (отрицание)     |
| `A ∧ B`  | И (AND)            |
| `A ∨ B`  | ИЛИ (OR)           |
| `A ⊕ B`  | Исключающее ИЛИ (XOR) |
| `A ↓ B`  | NOR                |
| `A ↑ B`  | NAND               |
| `A → B`  | Импликация         |
| `A ↔ B`  | Эквиваленция       |

---

## 🛠 Требования

- Python 3.x  
- Модуль `tkinter` (входит в стандартную библиотеку Python)
