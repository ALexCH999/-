# -
Калькулятор логических функций (на Python)
Описание:
Это простое графическое приложение, написанное на Python с использованием библиотеки Tkinter, которое позволяет строить таблицу истинности для произвольных логических выражений.

Пользователь может ввести логическое выражение, используя стандартные булевы операторы (∧, ∨, ⊕, ~, →, ↔, ↑, ↓ и др.), и программа автоматически сгенерирует таблицу истинности, отображая значения всех подвыражений и итогового результата.

Примеры операторов:
+--------------------------------+
| Оператор  |	     Описание      |
+--------------------------------|
| ~A	      |   НЕ A (отрицание) |
| A ∧ B	    |   И                |
| A ∨ B	    |   ИЛИ              |
| A ⊕ B	  |   Исключающее ИЛИ  |
| A ↓ B	    |   NOR              |
| A ↑ B	    |   NAND             |
| A → B	    |   Импликация       |
| A ↔ B	    |   Эквиваленция     |
+--------------------------------+

Требования:
    Python 3.x
    Стандартная библиотека tkinter (обычно включена по умолчанию)
